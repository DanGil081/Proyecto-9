Diseño web adaptable 

Dado que más del 50% de todo el tráfico web proviene de un dispositivo móvil, la creación de un sitio web optimizado para dispositivos móviles ya no es una opción, es una necesidad. Todo comienza con un diseño web receptivo.
El diseño web receptivo permite que su sitio web se adapte y brinde una experiencia óptima en cualquier dispositivo desde el que se esté viendo. Esto significa que su sitio web se verá genial y funcionará perfectamente para un usuario en una computadora de escritorio, portátil, tableta o teléfono inteligente

Alojamiento Web y Mantenimiento

En Proyecto 9 contamos con paquetes de Alojaminto de paginas web con buena seguridad además de ofrecer un servicio de mantenimiento para el tamaño y las necesidades de su empresa. ¿Solo busca mantener y realizar pequeñas actualizaciones en su pequeño sitio web existente? El paquete básico puede ser el más adecuado para usted. Alternativamente, si no solo busca mantenimiento, sino también mejora continua y optimización de motores de búsqueda para su sitio web de tamaño mediano a grande, los paquetes Mejorados serán perfectos.

Si tiene necesidades mayores, contáctenos y podemos armar un paquete personalizado o realizar actualizaciones por hora según sea necesario.

Datos personales.- Cualquier información concerniente a una persona física identificada o identificable.

Titular.- La persona física (TITULAR) a quien identifica o corresponden los datos personales.

Responsable.- Persona física o moral de carácter privado que decide sobre el tratamiento de los datos personales.

Tratamiento.- La obtención, uso (que incluye el acceso, manejo, aprovechamiento, transferencia o disposición de datos personales), divulgación o almacenamiento de datos personales por cualquier medio.

Transferencia.- Toda comunicación de datos realizada a persona distinta del responsable o encargado del tratamiento.

Derechos ARCO.- Derechos de acceso, rectificación, cancelación y oposición.
